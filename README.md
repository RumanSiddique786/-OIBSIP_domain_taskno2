#  Unemployment Analysis in India (Covid-19 Impact)

##  Project Overview

This project analyzes the unemployment rate in India during the year 2020, with a special focus on the impact of the Covid-19 pandemic.  

Unemployment rate represents the percentage of unemployed individuals in the total labor force. During the Covid-19 lockdown period, India experienced a significant economic slowdown, which affected employment across multiple regions.

The purpose of this project is to explore the data, identify trends, and visualize how unemployment changed before and during the pandemic.

---

##  Objective

The main objectives of this project are:

- To analyze unemployment trends in India during 2020  
- To compare unemployment rates before and during Covid-19  
- To identify regions most affected by unemployment  
- To visualize patterns using graphical analysis  

---

- **Source:** Kaggle

##  Steps Performed

1. Downloaded dataset using KaggleHub.
2. Loaded dataset into a Pandas DataFrame.
3. Cleaned column names and removed duplicate columns.
4. Converted the Date column into datetime format.
5. Generated statistical summary of the dataset.
6. Split data into:
   - Pre-Covid period (Before March 2020)
   - Covid period (March 2020 onwards)
7. Created visualizations:
   - Overall unemployment trend
   - Covid-period unemployment trend
   - Pre vs During Covid comparison
   - Region-wise unemployment analysis

---

##  Key Visualizations

- Line plot of unemployment rate over time  
- Covid-period unemployment trend analysis  
- Comparison between pre-Covid and Covid periods  
- Region-wise average unemployment rate  

---

##  Key Insights

- A sharp increase in unemployment was observed after March 2020.
- The peak unemployment rate occurred during mid-2020 lockdown period.
- Gradual recovery was observed in later months.
- Some regions were more affected than others.
